# Documentación de Laravel

Puedes entrar la documentacion de laravel original en [https://laravel.com/docs](https://laravel.com/docs)

## Reglas para contribuir
Si estás subiendo la documentación para la **versión estable**, envialo a su rama correspondiente. Por ejemplo, la documentación para Laravel 8 debe ser subida la rama `8.x`. La documentación integrada en la proxima versión debe ser enviada a la rama `master`
